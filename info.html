About PIEFACE

PIEFACE (Personalized Interactive Environment For Automata Combination Exploration) is a tool for visualizing and debugging gadget-based reductions in theoretical computer science. It was built to help understand reinforcement learning (RL) agents trained to synthesize motion-planning gadget constructions.

What Are Gadget Reductions?

In complexity theory, gadgets are modular components used in reductionsâ€”like logic gates for computational hardness proofs. These gadgets help encode constraints in problems like Sokoban, PushPush, or block-sliding puzzles.

Common gadgets include:

  • AP2T: Anti-Parallel 2-Toggle
  • C2T: Crossing 2-Toggle
  • L2T: Locking 2-Toggle
  • NWT: Noncrossing-Wire Toggle,
  • And others used in PSPACE-hardness proofs

Purpose of PIEFACE

Originally developed to help debug RL-discovered gadget simulations, PIEFACE lets users step through traces, view gadget states, and interact with known constructions.

Current Features

  • Step-by-step visualization of agent-discovered traces
  • Preloaded examples (e.g. AP2T to C2T)
  • Support for all 4-location gadget types
